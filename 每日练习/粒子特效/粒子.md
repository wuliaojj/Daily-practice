# 实现页面粒子背景

要实现页面粒子的背景，其实很简单，我们只需要将particles.js引入到html里面，然后根据particles.js的文档，添加配置项就行了。

```html
    <script src='./particles.min.js'></script>      //引入particles.min.js插件
    <script src='./config.js'></script>             //config.js是我们写配置项的文件
```

然后我们需要用一个载体来告诉配置项作用的区域。
```html
    <div id="particles-js"></div>       //这里的id要和配置项里的指向一样
```
