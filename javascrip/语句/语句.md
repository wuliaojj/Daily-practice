###break和continue语句
```
//break和continue用于在循环中精确地控制代码的执行。
//break会立即退出循环体
//continue也会退出本次循环，继续从下次循环开始

var num
for(var i=1;i<10,i++){
    if(i%5==0){
        break
    }
    num++
}
alert(num)  //4

for(var i=1;i<10,i++){
    if(i%5==0){
        continue
    }
    num++
}
alert(num)  //8
```
